pkgctl-db-move(1)
=================

Name
----
pkgctl-db-move - Move packages between binary repositories.

Synopsis
--------
pkgctl db move [OPTIONS] [SOURCE_REPO] [TARGET_REPO] [PKGBASE]...

Description
-----------

Move packages between binary repositories i.e. from 'extra-testing' to 'extra'.

Options
-------

*-h, --help*::
	Show a help text
